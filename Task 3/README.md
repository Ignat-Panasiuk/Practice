## Краткое описание реализованного функционала
Простейшее однооконное (SDI) приложение для создания и редактирования единичной карточки клиента фитнес-клуба с использованием главного меню программы.

* Управление через меню: все основные операции (открытие, сохранение, добавление и удаление) вынесены в удобное системное меню «Файл» и «Правка».
* Работа с файлами: реализовано сохранение и загрузка информации о клиенте в двух форматах: текстовом и бинарном.
* Журнал действий (Логгер): программа ведет непрерывную запись всех действий пользователя, выводя их на экран и сохраняя во внешний текстовый файл-журнал.
* Сохранение настроек: программа запоминает размер и положение окна на экране, а также путь к последнему открытому файлу и восстанавливает их при следующем запуске.
* Защита от потери данных: при попытке закрыть программу с несохраненными изменениями пользователю выводится модальное окно с запросом на подтверждение действий.
