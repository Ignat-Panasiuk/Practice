## Краткое описание реализованного функционала
Приложение предназначено для удобного ведения и редактирования базы данных клиентов фитнес-клуба в виде таблицы. 

* Управление базой: пользователь может добавлять новых клиентов, удалять существующие записи и редактировать информацию напрямую в ячейках таблицы.
* Сохранение и загрузка: база данных может быть сохранена и загружена из файлов в двух форматах: .CSV и .JSON.
* Поиск и фильтрация: реализован быстрый поиск клиентов по ФИО и фильтрация таблицы по тарифам абонементов.
* Сортировка: встроенная сортировка данных по любой колонке таблицы.
* Автоматическая статистика: при любых изменениях базы приложение мгновенно пересчитывает и выводит на экран статистику.
* Цветовая подсветка: строки клиентов, у которых осталось менее трех визитов, автоматически подсвечиваются красным цветом для привлечения внимания оператора.
